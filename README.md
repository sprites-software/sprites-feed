# Social Feed by SPRITES

## Installation, Activation
To Install and Activate this plugin, move the folder to your `wp-content/plugins` directory and activate the plugin via Wordpress administration.

## Dependency install
```
composer install
```

## For developers
```
composer install
yarn install
yarn watch
```
